T-Call Control

This project leverages the TTGO T-Call ESP32 module, a robust IoT development board combining the ESP32 microcontroller with the SIM800 modem for cellular communication. It demonstrates basic interactions with the SIM800 modem using AT commands, enabling functionalities such as making calls, sending SMS, checking battery status, and managing received messages.
